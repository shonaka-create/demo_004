## スクロール演出・アニメーション方針

| 演出 | 対象 | 実装方法 |
|------|------|----------|
| フェードインアップ | 各セクション見出し | CSS `@keyframes` + Intersection Observer |
| カウントアップ | SECTION 07 数値 | JavaScript（軽量ライブラリ or バニラ） |
| パーティクル | HERO背景 | CSS アニメーション（JS不使用推奨） |
| スムーススクロール | 全体 | `scroll-behavior: smooth` |
| セクション背景切替 | 全体 | 背景色は各セクションに直接指定 |
