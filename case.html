Network Integration
Case 01 — Network Integration
東北エリア・社会福祉法人4法人のネットワーク統合と経営標準化プロジェクト

高齢化が急速に進む東北エリアで、それぞれ単独で経営する4つの社会福祉法人が、人材不足と財務悪化の二重苦に直面。各法人の強みと弱みを分析し、機能分担と共有バックオフィスの設計を行いました。

共通の人材プールを形成し、繁忙期・閑散期での柔軟な人員配置を実現。法人ごとに抱えていた経理・労務業務を統合することで、管理コストを大幅に削減しました。

Result
管理コスト 32%削減 / 職員定着率 18%向上 / 4法人合計で黒字転換を実現
02
DX Implementation
Case 02 — DX Implementation
特別養護老人ホーム向け介護記録DXシステムの設計・導入支援

紙ベースの介護記録と手書きの申し送りノートが常態化していた特別養護老人ホームにおいて、現場スタッフの実態調査から始まる段階的なDX推進を実施しました。

現場の抵抗感を最小化するため、システム設計・研修・定着支援を一体的に提供。導入6ヶ月後の使用率90%超を達成し、夜勤時の記録負担を大幅に軽減しました。

Result
記録業務時間 45%削減 / 申し送りミス 80%減 / 職員満足度スコア 26pt向上
03
Management Reform
Case 03 — Management Reform
慢性的赤字法人の経営構造改革——3年間で黒字転換を実現した組織再設計

5年連続赤字で理事会が経営危機を認識していた社会福祉法人において、財務・組織・サービス設計の3面からの構造改革を実施。問題の根本が「理事長の属人的経営」にあることを特定し、意思決定システムの再設計から着手しました。

組織図の刷新、人事評価制度の導入、財務ダッシュボードの整備を段階的に実施。3年目には黒字転換を達成し、法人の継続性を確保しました。

Result
3年間で黒字転換 / 職員離職率 42%→18%に改善 / 新規事業開始による収益多角化を実現
