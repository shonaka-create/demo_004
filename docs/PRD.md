# フェンリルイノベーションズ株式会社 — コーポレートサイト PRD

## 1. 概要

### 1.1 ワンライナー
社会福祉業界（医療・介護・保育）を対象とした総合コンサルティング会社のコーポレートサイト。潜在クライアント・採用候補者・メディア向けに、会社の思想・事業・実績を伝える。

### 1.2 背景・課題
- **誰が**: フェンリルイノベーションズ株式会社（2025年設立）
- **どんな状況で**: スタートアップ段階でブランド認知がゼロ
- **何に困っているか**: 「社会福祉のインフラを再設計する」という高い理念を、初見の訪問者に短時間で伝えられるサイトがない

### 1.3 解決策
ハイエンドな企業サイト（ダーク系・Gold アクセント）で思想・事業・実績を構造的に伝え、問い合わせ・採用応募への動線を確保する。

## 3. ユーザーストーリー

### 3.1 必須（MVP）

- [ ] 潜在クライアントとして、会社の事業概要を3分以内に理解したい。なぜなら、問い合わせ前に会社の信頼性を確認したいから。
  - 受け入れ基準: TOP → ABOUT/SYSTEM DESIGN → BUSINESS → CONTACT の動線が機能している
- [ ] 潜在クライアントとして、問い合わせフォームから相談内容を送信したい。なぜなら、まず対話から始めたいから。
  - 受け入れ基準: Contact Form 7 が正常送信・自動返信メールが届く
- [ ] 採用候補者として、会社の理念と募集職種を確認したい。なぜなら、応募するかどうか判断したいから。
  - 受け入れ基準: RECRUIT ページに採用メッセージ・求める人物像・職種カードがある
- [ ] 全ユーザーとして、スマートフォンで快適に閲覧したい。
  - 受け入れ基準: 375px以上の画面幅で崩れがない

### 3.2 Nice-to-have（v2以降）

- [ ] INSIGHT ブログで業界知見を読みたい（WordPressで動的投稿）
- [ ] CASE 導入事例でフェンリルの実績を確認したい（ACFカスタムフィールド）
- [ ] Google Analytics でサイト訪問データを確認したい

## 5. 非機能要件

| カテゴリ | 要件 |
|---------|------|
| パフォーマンス | Google PageSpeed Insights スコア 80以上（モバイル） |
| SEO | 各ページのメタタイトル・ディスクリプション設定（SEO SIMPLE PACK） |

| アクセシビリティ | コントラスト比 WCAG AA 基準（4.5:1以上）・タップターゲット 44px以上 |
| レスポンシブ | 375px（iPhone SE）〜 1280px以上を対象 |

## 6. 技術スタック

| レイヤー | 技術 | 選定理由 |
|---------|------|---------|
| フロントエンド（デモ） | HTML / CSS / JavaScript（バニラ） | デモ段階でフレームワーク不要 |
| フロントエンド（本番） | WordPress + Emanon PREMIUM | クライアントが管理しやすいCMS |
| スタイリング | カスタムCSS（CSS変数使用） | Tailwind不使用・Emanon互換 |
| CMS | WordPress 6.x | 既存ホスティング環境 |
| テーマ | Emanon PREMIUM + 子テーマ | WordPress実装の基盤 |
| プラグイン | Contact Form 7 / ACF / Custom Post Type UI / SEO SIMPLE PACK / WP Super Cache | 各機能を担当 |
| ホスティング | さくらインターネット（共有サーバー） | クライアント既存環境 |
| ドメイン | 取得予定（f-innov.jp 系推奨） | — |
| SSL | Let's Encrypt（さくら無料SSL） | — |

## 10. リスクと対策

| リスク | 影響度 | 発生確率 | 対策 |
|--------|--------|---------|------|
| Emanon PREMIUM との CSS競合 | 高 | 中 | 子テーマのみでカスタマイズ・`!important` 最小化 |
| 会社情報（所在地・代表者）が未確定 | 中 | 高 | プレースホルダーで対応・登記完了後に更新 |
| Contact Form 7 スパム | 中 | 高 | reCAPTCHA v3 導入 |
| さくらサーバーのPHP版本 | 中 | 低 | PHP 8.1以上を確認 |
| 画像素材（代表者写真等）未入手 | 低 | 高 | 抽象ビジュアルプレースホルダーで対応済み |
