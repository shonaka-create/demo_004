# フェンリルイノベーションズ — ユーザーフロー図

## 1. サイト全体ナビゲーションフロー

```mermaid
flowchart TD
    A[サイト訪問\n検索・直接・紹介] --> B{入口ページ}

    B -->|TOP| C[index / TOP]
    B -->|検索流入| D[各ページ直接着地]

    C --> E{関心ページへ}
    E -->|会社を知りたい| F[ABOUT]
    E -->|事業内容| G[BUSINESS]
    E -->|思想・構想| H[SYSTEM DESIGN]
    E -->|知見・記事| I[INSIGHT]
    E -->|実績を見たい| J[CASE]
    E -->|採用情報| K[RECRUIT]
    E -->|問い合わせ| L[CONTACT]

    F --> M{次の行動}
    G --> M
    H --> M
    I --> N[記事詳細]
    N --> M
    J --> O[事例詳細]
    O --> M
    K --> M

    M -->|問い合わせる| L
    M -->|さらに回遊| E
    M -->|離脱| Z[サイト離脱]

    L --> P[フォーム送信]
    P --> Q[サンクスメッセージ表示]
    Q --> Z
```

## 2. 問い合わせフロー

```mermaid
flowchart TD
    A[CONTACT ページ到達] --> B[お問い合わせ種別を選択]
    B --> C[フォーム入力\n氏名・メール・内容など]
    C --> D{バリデーション}
    D -->|エラー| E[エラーメッセージ表示]
    E --> C
    D -->|OK| F[個人情報同意チェック]
    F --> G{送信ボタン押下}
    G --> H[送信処理中\nローディング表示]
    H --> I{送信結果}
    I -->|成功| J[サンクスメッセージ表示]
    I -->|失敗| K[エラーメッセージ表示\n再送信を促す]
    J --> L[自動返信メール送信]
    J --> M[管理者通知メール送信]
    K --> G
```

## 3. INSIGHT ブログ閲覧フロー

flowchart TD

    C --> I[記事カード一覧\n3カラムグリッド]
    D --> I
    E --> I
    F --> I
    G --> I
    H --> I

    I --> J{ページネーション}
    J -->|次ページ| I
    I --> K[記事詳細ページ]

## 4. CASE 事例閲覧フロー

```mermaid
flowchart TD
    A[CASE 一覧ページ] --> B{業種フィルター}
    B -->|全て| C[全事例表示]
    B -->|介護| D[介護事例]
    B -->|医療| E[医療事例]
    B -->|保育| F[保育事例]
    B -->|宗教法人| G[宗教法人事例]

    C --> H[2カラム事例カード一覧]
    D --> H
    E --> H
    F --> H
    G --> H

    H --> I[事例詳細ページ]
    I --> J[課題→アプローチ→成果の読了]
    J --> K{次のアクション}

    K -->|一覧に戻る| A
```
