## タスク分解

### Task 4.1: ドメイン設定・DNS 紐付け
**見積もり**: 1h（DNS 反映待ち最大72時間）  
**依存**: ドメイン取得済み  
**内容**:
- さくらコントロールパネル > ドメイン設定 > ドメイン追加
- DNS の A レコードをさくらサーバーの IP に設定
- 反映確認（`nslookup` または `dig` コマンドで確認）

**参照**: [04_WordPress実装手順書.md](../../04_WordPress実装手順書.md) — STEP 10

**完了条件**:
- [ ] ドメインにアクセスして WordPress サイトが表示される

---

### Task 4.2: SSL 設定（Let's Encrypt）
**見積もり**: 30min  
**依存**: Task 4.1（DNS 反映後）  
**内容**:
- さくらコントロールパネル > SSL設定 > 無料SSL（Let's Encrypt）を適用
- WordPress アドレスを `https://` に変更
- `.htaccess` に HTTP→HTTPS リダイレクトを追加

**完了条件**:
- [ ] `https://` で表示される（鍵マーク確認）
- [ ] `http://` アクセス時に `https://` へリダイレクトされる
- [ ] Mixed Content エラーがない（全リソースが HTTPS）
